# Spring_jpa_mybatis_thymeleaf

##1. 설명
일반적인 SpringBoot JPA(Hibernate) Project 구성으로 구성하되, 상황에 따라 Mybatis를 사용할 수 있도록 구성해본다.

##2. Project 생성 환경
1. Mac OSX
2. IntelliJ IDEA
3. JDK 1.8

##3. 구성
1. Spring Boot
2. JPA (Hibernate)
3. Mybatis
4. Thymeleaf
5. H2 Database
6. Lombok

##4. 실행방법
1. `git clone https://github.com/wonzopein/spring_jpa_mybatis_thymeleaf.git`
2. `mvn spring-boot:run`
